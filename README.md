## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Setup

### Set .env

```shell
cp src/common/envs/.example.env src/common/envs/.development.env
```

## Installation docker

Create build
```shell
npm run dockerize
```

Run build
```shell
npm run start:container
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
